# TicTacToe Game with Python and KivyMD #

> Just a simple tictactoe game to show off how kivy works
> Feel free to edit
>
> Please Notice:
> you have to make sure kivymd is installed:
>
> ```CMD
> pip install kivymd
>```
>
>after installing kivymd just run the main.py with:
>
>```CMD
> python .\main.py
>```
